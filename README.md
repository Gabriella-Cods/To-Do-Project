# 📝 Lista Diária - Gerenciador de Tarefas

Meu projeto tem por objetivo que o usuário consiga cadastrar suas tarefas diárias, separá-las estrategicamente e de um jeito mais fácil que ajude em suas atividades do cotiadiano.


Este projeto foi desenvolvido para a disciplina de **Projeto de Software** sob orientação do Prof. Antonio de Oliveira Dias. O objetivo é criar uma aplicação Full Stack funcional dividida em sprints (ACs).

## 🚀 Tecnologias Utilizadas

Este projeto utiliza uma arquitetura de 3 camadas:

* **Front-end:** HTML5, CSS3 e JavaScript (ES6+).
* **Back-end:** Python com framework Flask.
* **Banco de Dados:** SQLite para persistência de dados.
